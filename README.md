# ESP32 多功能监测系统

## 项目简介

这是一个基于ESP32的多功能监测系统，集成了超声波测距、温度监测、时间显示和运动密度分析等功能。系统通过OLED显示屏展示各种信息，并通过旋转编码器进行菜单操作。

## 硬件要求

- ESP32开发板（ESP32-S3）
- JSN-SR04T 一体化超声波测距模块
- DS18B20 温度传感器
- 0.96英寸 OLED显示屏（I2C接口）
- 旋转编码器（带按键）
- 蜂鸣器
- LED
- 面包板和连接线

## 软件依赖

- ESP-IDF 5.0+
- 标准C库
- FreeRTOS

## 功能说明

### 模式1：实时监测
- 显示当前时间（24小时制）
- 显示DS18B20温度传感器读数
- 显示超声波传感器测量的距离
- 实时更新数据

### 模式2：图片显示
- 显示一个简单的16x16像素图片
- 提供按键退出功能

### 模式3：30秒运动密度分析
- 启动30秒的超声波采样
- 实时显示当前距离和倒计时
- 分析运动物体的密度和运动变化
- 显示分析结果（密度等级和运动变化等级）

### 模式4-10：预留模式
- 目前仅显示模式名称
- 等待后续功能扩展

## 使用方法

1. **硬件连接**：
   - 超声波传感器：TRIG -> GPIO 4, ECHO -> GPIO 5
   - OLED显示屏：SDA -> GPIO 21, SCL -> GPIO 22
   - 旋转编码器：CLK -> GPIO 18, DT -> GPIO 19, SW -> GPIO 23
   - DS18B20：DATA -> GPIO 14
   - 蜂鸣器：GPIO 2
   - LED：GPIO 15

2. **编译烧录**：
   - 使用ESP-IDF命令行工具或VS Code插件编译项目
   - 烧录到ESP32开发板

3. **操作说明**：
   - 旋转编码器：选择菜单模式
   - 按下编码器按键：进入选中的模式
   - 在模式中按下按键：退出当前模式，返回菜单

## 系统特性

- **模块化设计**：各功能模块独立封装，便于维护和扩展
- **实时数据**：时间、温度、距离等数据实时更新
- **运动分析**：基于超声波的运动密度分析
- **用户友好**：OLED显示清晰，操作简单直观
- **资源管理**：自动管理系统资源，避免泄漏

## 注意事项

1. **超声波传感器**：
   - 测量范围：2cm - 600cm
   - 避免在强干扰环境下使用
   - 确保传感器前方无障碍物

2. **温度传感器**：
   - 测量范围：-55°C 到 +125°C
   - 需要在数据线上添加4.7kΩ上拉电阻

3. **OLED显示屏**：
   - 分辨率：128x64
   - 使用I2C通信，地址默认0x3C

4. **网络功能**：
   - 本版本已移除天气功能，避免TLS连接问题
   - 保留了网络初始化代码，可根据需要重新启用

## 故障排除

- **OLED不显示**：检查I2C连接和地址
- **超声波测量错误**：检查传感器连接和电源
- **菜单操作无响应**：检查编码器连接
- **温度显示异常**：检查DS18B20连接和上拉电阻
